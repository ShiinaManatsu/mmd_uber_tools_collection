# MMD Uber Tools Collection

用于MMD流程的Blender工具集.

## 包含工具

### PMX 动画代理

原插件名字叫做"IKFix", 通过烘焙 PMX IK 物理为 Alembic 文档, 从而在 Blender 中获得和在 MMD 中相同的 IK 表现.
修改后的.

修改后的插件将保存所有骨骼, 从而在 Blender 中获得与在 MMD 中完全一致的表现

源代码: [bilibili](https://www.bilibili.com/video/BV1Za4y1C7rs/)
